# 课程设计-课程管理系统

## 项目简介
课程管理系统是基于Java Swing 和Mysql数据库的课程管理。
包含：
1、课程查询  2、课程添加  3、课程删除 4、课程更新等功能
2、对Mysql数据库的课程表进行增删改查
